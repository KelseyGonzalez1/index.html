Explanation of AJAX

AJAX (Asynchronous JavaScript and XML) makes websites work smoothly. It allows web pages to talk to servers without waiting for the entire page to reload. In our assignment, when you hit the "Get CD collection" button, AJAX uses something called XMLHttpRequest to ask the server for CD details. The server responds, and with the help of JavaScript, we can neatly display this CD information in a table on the webpage.

When we look closer at how AJAX works on this page, we start by creating an XMLHttpRequest object -the messenger between the webpage and the server. We then instruct it on what to do when the server responds , like updating the table. The beauty of AJAX lies in its asynchronous operation, it gets things done without holding up the whole webpage. With AJAX the webpage looks and works great.
